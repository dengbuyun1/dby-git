# 树莓派TCP模块测试指南

## 📁 文件说明

### 1. `tcp_module.py` - TCP服务器模块
核心功能模块，包含：
- `TCPServer` 类：处理TCP连接、数据接收、胰岛素计算
- `create_tcp_server()` 函数：快速创建服务器
- 可独立运行测试

### 2. `main_test.py` - 测试主程序
完整的测试程序，适合在树莓派上运行：
- 启动TCP服务器
- 显示接收到的数据（表格形式）
- 支持GPIO控制（可选）
- 实时统计

### 3. `test_tcp_module.py` - 快速测试脚本
本地测试脚本，可以在PC上运行：
- 自动启动服务器和客户端
- 发送模拟数据
- 验证响应是否正确

## 🐛 故障排除

### 问题1：服务器启动失败
```
✗ TCP服务器启动失败: [Errno 98] Address already in use
```

**解决方法：**
```bash
# 查找占用端口的进程
lsof -i :5000
# 或
netstat -tulpn | grep 5000

# 杀死进程
kill -9 进程ID
```

### 问题2：PC无法连接树莓派
**检查清单：**
1. 确认树莓派IP地址正确：`hostname -I`
2. 确认防火墙允许5000端口
3. 确认两者在同一网络
4. 尝试ping：`ping 树莓派IP`

### 问题3：收不到数据
**检查：**
1. PC端simulator.py的IP配置是否正确
2. 树莓派程序是否正在运行
3. 网页是否点击了"开始仿真"

## 📝 开发笔记

### TCP模块特点
- ✓ 线程安全的服务器
- ✓ 支持多客户端（每个连接一个线程）
- ✓ 自动重连容错
- ✓ 实时数据更新
- ✓ 简单的胰岛素计算算法

### 与完整版的区别
- 简化版：只包含基础的TCP通信和简单算法
- 完整版（`tcp_server_module.py`）：包含IOB/COB计算、数据库存储等
