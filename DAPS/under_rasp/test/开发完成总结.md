## 📊 胰岛素计算公式

当前使用的简化算法：

```python
# 基础胰岛素（恒定）
basal = 0.8 U

# 血糖校正
bg_error = cgm - 110  # 偏差
correction = bg_error / 50  # 校正量

# 餐食补偿
meal_bolus = cho / 12  # 碳水化合物转换

# 总大剂量
bolus = max(0, correction + meal_bolus)

# 总胰岛素
insulin = basal + bolus
```

**示例：**
- 血糖125, 无餐食 → 校正0.3U → 总1.1U
- 血糖112, 30g碳水 → 校正0.04U + 餐食2.5U → 总3.34U
- 血糖88, 无餐食 → 校正-0.44U（为0） → 总0.8U（仅基础）

## 🔧 可调参数说明

在 `tcp_module.py` 的 `TCPServer.__init__()` 中：

| 参数 | 默认值 | 说明 | 调整建议 |
|------|--------|------|----------|
| `TARGET_BG` | 110.0 | 目标血糖 (mg/dL) | 正常范围：70-120 |
| `BASAL_BASE` | 0.8 | 基础胰岛素 (U) | 根据患者体重调整 |
| `KP` | 0.01 | 比例系数 | 暂未使用 |
| `CARB_RATIO` | 12.0 | 碳水比率 (g/U) | 每12g碳水需1U胰岛素 |
| `CORRECTION_FACTOR` | 50.0 | 校正因子 | 每50mg/dL偏差需1U |
