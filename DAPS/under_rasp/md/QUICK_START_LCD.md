# LCD显示快速使用指南

## 📺 LCD显示说明

### 显示布局 (2行×16字符)
```
┌────────────────┐
│M:AUTO SPD: 75%│  ← 第1行: 模式 + 速率
│IOB:3.2   14:23│  ← 第2行: IOB + 时间
└────────────────┘
```

---

## 🎮 按钮功能

| 按钮 | 功能 | LCD变化 |
|------|------|---------|
| **Button1** | 紧急停止 | 显示 "EMERGENCY STOP!" |
| **Button2** | 模式切换 | `M:AUTO` ↔ `M:MANU` |

---

## 🔄 两种工作模式

### AUTO模式 (自动)
```
┌────────────────┐
│M:AUTO SPD: 75%│  ← 显示 AUTO
│IOB:3.2   14:23│
└────────────────┘
```
- ✅ 自动计算并注射胰岛素
- ✅ LED绿灯闪烁(注射时)
- ✅ 电机自动运行

### MANUAL模式 (手动)
```
┌────────────────┐
│M:MANU SPD:  0%│  ← 显示 MANU
│IOB:0.0   09:15│
└────────────────┘
```
- ✅ 显示计算结果
- ❌ **不**自动注射
- ⚠️ 需手动控制

---

## 📊 显示信息解读

### 第1行信息

#### 控制模式
- `M:AUTO` → 自动模式(系统控制)
- `M:MANU` → 手动模式(人工控制)

#### 电机速率
- `SPD:  0%` → 停止
- `SPD: 50%` → 中速
- `SPD:100%` → 全速

### 第2行信息

#### IOB值 (体内活性胰岛素)
- `IOB:0.0` → 无活性胰岛素
- `IOB:2.5` → 剩余2.5U
- `IOB:10.0` → 剩余10.0U(高值)

#### 时间
- `12:34` → 12:34(24小时制)
- `09:15` → 上午9:15
- `22:47` → 晚上10:47

## 🚀 快速操作

### 1️⃣ 启动系统
```bash
python3 rasp_integration.py
```

初始显示:
```
┌────────────────┐
│M:AUTO SPD:  0%│
│Wait TCP...     │
└────────────────┘
```

### 2️⃣ TCP连接后
```
┌────────────────┐
│M:AUTO SPD:  0%│
│IOB:0.0   14:23│  ← 显示实时数据
└────────────────┘
```

### 3️⃣ 切换到手动模式
**操作**: 按下 Button2

**显示变化**:
```
┌────────────────┐
│M:MANU SPD:  0%│  ← AUTO变成MANU
│IOB:0.0   14:23│
└────────────────┘
```

### 4️⃣ 自动注射中
(仅AUTO模式)
```
┌────────────────┐
│M:AUTO SPD: 75%│  ← 速率>0%
│IOB:3.2   14:25│  ← IOB值增加
└────────────────┘
```

### 5️⃣ 紧急停止
**操作**: 按下 Button1

**显示**:
```
┌────────────────┐
│EMERGENCY STOP! │  ← 紧急停止消息
│                │
└────────────────┘
```

## ⚠️ 常见状态

### TCP未连接
```
┌────────────────┐
│Wait TCP...     │
│                │
└────────────────┘
```
**含义**: 等待PC端连接

### TCP超时
```
┌────────────────┐
│TCP Timeout...  │
│                │
└────────────────┘
```
**含义**: 通信超时,检查网络

### 正常运行
```
┌────────────────┐
│M:AUTO SPD: 50%│
│IOB:2.3   16:45│
└────────────────┘
```
**含义**: 系统正常工作

## 🔍 故障诊断

### LCD无显示
1. 检查I2C连接
2. 运行 `i2cdetect -y 1`
3. 检查电源(5V)

### 显示乱码
1. 重启系统
2. 检查I2C地址
3. 更新LCD驱动

### 模式切换无效
1. 检查Button2连接
2. 查看日志文件
3. 测试按钮功能
