## 🔄 数据流程图

```
PC端仿真数据
    │
    ├─→ bg (血糖)
    ├─→ cgm (CGM读数)
    └─→ cho (碳水摄入)
    │
    ↓ [TCP发送]
    │
树莓派TCP服务器接收
    │
    ↓ [数据解析]
    │
存储到data_storage
    │
    ↓ [传递数据]
    │
算法模块计算
    │
    ├─→ 计算IOB (体内活性胰岛素)
    ├─→ 计算COB (未吸收碳水)
    ├─→ 计算basal (基础率)
    ├─→ 计算bolus (餐时大剂量)
    └─→ 汇总insulin (总剂量)
    │
    ↓ [控制指令]
    │
    ├─→ 发送到TCP客户端 (返回PC)
    │
    └─→ 发送到电机控制器
        │
        ↓ [转换为步数]
        │
        ├─→ 计算PWM频率
        ├─→ 生成PWM信号
        └─→ 驱动TB6600
            │
            ↓
        步进电机转动
            │
            ↓
        推注胰岛素
```

## 📊 控制算法流程

```
输入: bg, cgm, cho
│
├─→ [步骤1] 计算IOB
│   │
│   └─→ 遍历胰岛素历史
│       └─→ 双指数衰减模型
│           └─→ IOB = Σ[Dose × (0.65×e^(-t/1.2h) + 0.35×e^(-t/3h))]
│
├─→ [步骤2] 计算COB
│   │
│   └─→ 遍历碳水历史
│       └─→ 抛物线吸收模型
│           └─→ COB = Σ[CHO × max(0, 1-t/3h)²]
│
├─→ [步骤3] 计算基础率
│   │
│   └─→ basal = basal_base + Kp × (bg - target_bg)
│
├─→ [步骤4] 计算餐时剂量
│   │
│   ├─→ 碳水覆盖 = cho / carb_ratio
│   ├─→ 高血糖校正 = (bg - target) / correction_factor
│   ├─→ IOB抑制 = IOB × iob_sensitivity
│   └─→ COB增强 = COB × cob_boost / carb_ratio
│       │
│       └─→ bolus = 碳水覆盖 + 校正 - IOB抑制 + COB增强
│
└─→ [步骤5] 汇总
    │
    └─→ insulin = basal + bolus
        │
        └─→ 输出: insulin, basal, bolus
```

## 🎯 电机控制流程

```
输入: insulin (U)
│
├─→ [转换1] 胰岛素 → 体积
│   volume_ml = insulin / concentration
│
├─→ [转换2] 体积 → 位移
│   distance_mm = (volume_ml × 1000) / syringe_area
│
├─→ [转换3] 位移 → 步数
│   steps = distance_mm / step_distance
│
├─→ [转换4] 步数 → 频率
│   frequency = steps / time_step
│   frequency = clamp(frequency, min_freq, max_freq)
│
└─→ [执行] PWM输出
    │
    ├─→ 设置频率: pwm.ChangeFrequency(frequency)
    ├─→ 设置占空比: pwm.ChangeDutyCycle(50%)
    └─→ 步进电机转动
```

## 📈 性能优化策略

```
优化点                    措施
│
├─ 通信延迟            → TCP长连接 + 缓冲
│
├─ 算法效率            → 历史数据限制 (deque)
│
├─ 电机响应            → 20Hz控制循环
│
├─ 显示刷新            → 0.5秒刷新率
│
└─ 内存占用            → 历史记录上限 (1000条)
```
